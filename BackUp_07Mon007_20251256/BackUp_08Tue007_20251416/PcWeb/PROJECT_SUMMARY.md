## 🚀 개발 환경

### 설치 및 실행
```bash
# 의존성 설치
npm install

# 개발 서버 시작
npm run dev

# 또는 Python 서버 사용
npm start
```

### 사용 가능한 명령어
- `npm run dev`: Live Server로 개발 서버 실행
- `npm start`: Python HTTP 서버 실행
- `npm run backup`: 데이터 백업
- `npm run restore`: 데이터 복원
- `npm run deploy`: Firebase 배포
