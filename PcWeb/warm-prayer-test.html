🎨 시안 A: 미니멀 카드 스타일 특징

  • 깔끔한 디자인: 순수한 흰색 배경과 미세한 드롭 섀도우
  • 현대적 타이포그래피: Pretendard 산세리프 폰트 사용
  • 미니멀 레이아웃: 불필요한 장식 요소 제거
  • 부드러운 애니메이션: 자연스러운 페이드인과 호버 효과
  • 접근성: 다크 모드 지원 및 모션 감소 옵션
  • 반응형: 모바일 환경에서도 최적화된 표시

⚙️ 주요 기능

자동 순회

등록된 선교사들의 기도 요청을 4초마다 자동으로 순회하며 표시합니다.

마우스 오버 제어

팝업에 마우스를 올리면 자동 순회가 일시정지되고, 마우스를 떼면 재개됩니다.

닫기 버튼

팝업 우상단의 × 버튼을 클릭하여 팝업을 닫고 순회를 재개할 수 있습니다.

실시간 데이터

Firebase에서 실시간으로 선교사 정보와 기도 요청을 가져옵니다.

지능형 위치

마커 위치에 맞춰 팝업을 표시하고, 화면 경계를 벗어나지 않도록 자동 조정합니다.

부드러운 전환

팝업 간 전환 시 자연스러운 애니메이션으로 사용자 경험을 향상시킵니다.

💡 사용 방법

  • 샘플 팝업 표시: 테스트용 샘플 팝업을 즉시 표시합니다
  • 순회 시작: 모든 선교사의 기도 요청을 자동으로 순회합니다
  • 순회 중지: 자동 순회를 중지하고 현재 팝업을 제거합니다
  • 팝업 지우기: 화면의 모든 팝업을 즉시 제거합니다
  • 기도 안내 팝업 테스트: 단일 기도 안내 팝업을 표시합니다
  • 기도 클릭 테스트 동시 10개: 10명의 선교사에 대한 기도 안내 팝업을 큐에 추가하여 순차 처리를 테스트합니다
  • 큐 상태 확인: 현재 기도 안내 팝업 큐의 상태를 확인합니다
  • 큐 초기화: 기도 안내 팝업 큐를 초기화하고 현재 팝업을 제거합니다
